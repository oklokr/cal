  예) 프레임: Matte Gray, 필름: Charcoal Gray, 투과율: 2%, 옵션: 기본 싱글 (2111)
  예) 프레임: Warm White, 필름: Charcoal Gray, 투과율: 7%, 옵션: 기본 싱글 (1121)
